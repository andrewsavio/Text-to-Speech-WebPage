# Text-to-Speech-WebPage
> ⚠️ This project is source-visible but licensed for private use only. Public deployment, SaaS usage, and commercial use are prohibited.

Web-based Text-to-Speech Platform powered by Pocket-TTS, featuring a serverless frontend (Netlify/Vercel) and a scalable FastAPI backend for real-time voice generation. Designed for AI demos, EdTech, accessibility tools, and SaaS-ready deployment.
